## Layout Principles

- User-facing personal feed, not admin dashboard.
- First screen should answer: what is worth my attention today?
- Every item should offer lightweight feedback.
- Concerts get special treatment because timing matters.
- Trends should be grouped by usefulness, not source.

## Core Feedback Actions

- `Track closely`
- `Save`
- `Not relevant`
- later: `More like this`, `Less like this`, reason chips
